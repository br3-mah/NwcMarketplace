Authentication

Phone Authentication

POST
/signin/{role}/phone
Initiate phone number sign-in process
No authentication required

Parameters

ParameterTypeRequiredDescription
rolestringYesRole: admin, user, vendor
phonestringYesPhone number

Request Example

{ "phone": "+1234567890" }

Response (200)

{ "message": "Welcome back! OTP sent to +1234567890", "data": { "user_id": 1, "channel": "phone", "role": "user", "identifier": "+1234567890", "expires_at": "2025-10-29T23:00:00.000000Z", "is_existing_user": true, "has_role": true }, "debug": { "otp": "123456" } }
POST
/verify/phone
Verify phone number with OTP
No authentication required

Parameters

ParameterTypeRequiredDescription
phonestringYesPhone number
otpstringYesOTP code

Response (200)

{ "message": "Phone number verified successfully.", "data": { "id": 1, "name": "John Doe", "fname": "John", "lname": "Doe", "gender": "male", "email": "john@example.com", "phone": "+1234567890", "phone_verified_at": "2025-10-29T22:40:00.000000Z", "email_verified_at": null, "roles": ["user"] }, "token": { "token_type": "Bearer", "access_token": "eyJ0eXAiOiJKV1QiLCJhbGciOiJSUzI1NiJ9..." } }
